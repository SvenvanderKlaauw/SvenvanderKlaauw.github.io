Het randapparatuur zijn alle onderdelen van een computer die niet onder het kernapparatuur vallen.
Het kernapparatuur van een computer bestaat uit het CPU en RAM.
Op een telefoon gaat dit om de apps die einstaleerd worden via de appstore.
Door middel van userinterfaces wordt er tussen de toepassingenlaag en de gebruiker gecommuniceert.
Dit gebeurt meestal door GUI's, wat een afkorting voor Graphical User Interface is.
Voorbeelden van die GUI's zijn touchscreen en het beeldscherm va een computer.
Alle handelingen die worden gedaan door een user wordt door het besturingssysteem omgezet naar events.
Die wordt vervolgens door event handlers in applicaties verwerkt.
Een event handler is een code dat een gebeurtenis verwerkt.
